Máxima Velocidad cambia tus imágenes destacadas al formato Webp, sin agregar CSS ni JS. No borra ninguna imagen, y siempre permite deshacer los cambios.

== Description ==

Máxima Velocidad es un plugin súper liviano, súper simple de usar que te permitirá mejorar el tiempo de carga de tu Wordpress, sin necesidad de conocimientos ténicos.
Toda su estrucutra se basa en las recomendaciones de PageSpeed Insights de Google, con lo que además de que tu sitio funcione más rápido, mejorarás tu SEO. 
Esta versión 1.1 te permite cambiar las imagenes destacadas de entradas y páginas (Thumbnail) al nuevo formato Webp. Obtendrás la misma calidad de imagen, con archivos más pequeños y sin agregar una linea de código que demore el tiempo de carga. *Siempre, siempre, pase lo que pase, podrás volver tu WordPress a cómo estaba antes del primer cambio.*
Seguiremos trabajando para agregar más funcionalidad con la premisa: simple, liviano, sin borrar nada, con la opción de deshacer. 

== Frequently Asked Questions ==

= Qué es el formato Webp =

Es un nuevo formato para imágenes, especial para la Web, diseñado para generar archivos más pequeños que descargan más rápido, sin perder calidad. 
Puedes ver un comparativo técnico en el sitio de desarrolladores de Google: https://developers.google.com/speed/webp/docs/webp_study


= Qué es PageSpeed Insights =

Es un proyecto destinado a mejorar la carga de los sitios Web. Te da un índice de tu velocidad de carga, tanto en escritorio como en móviles, y una lista de recomendaciones para mejorarla. 
El buscador de Google comparte el motoro de análisis con PageSpeed Insights, por lo que todo lo que mejores se reflejará en cómo Google te califica al indexarte. Mejorar la velocidad de carga mejora el SEO de tu WordPress.

== Screenshots ==
1. Cambiar las imágenes destacadas (Thumbnails) requiere solo clickear un botón. Y siempre se puede deshacer.
2. Máxima Velocidad detecta las imagenes que no se pueden cambiar: GIF animados y tanto GIF como PNG transparente. Para cada imagen informa el tamaño original y el nuevo, para que puedas conocer cuánto ganaste en peso de archivos.
